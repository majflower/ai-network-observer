## 🎯 Overview

Advanced network monitoring with **AI-powered threat detection**:
- 🤖 LLM analysis (Ollama)
- 🔍 DNS/HTTP/TLS deep inspection
- 📊 Real-time dashboards (Grafana)
- 🔗 Network graph (Neo4j)
- ⚡ Automated remediation

## 🚀 Quick Start
```bash
git clone [YOUR_REPO]
cd ai-network-observer/ai-network
docker-compose up -d
```

Access:
- Grafana: http://localhost:3000 (admin/networksecurity)
- Neo4j: http://localhost:7474 (neo4j/networksecurity)
